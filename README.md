# ChartsDemo
各种类型统计图 基本的线形图，柱形图，饼形图等

集成一个jar（ArcScroll.jar）包 包含了各种的统计图
基本使用
    <com.github.mikephil.charting.charts.LineChart
          android:id="@+id/line"
          android:layout_width="match_parent"
          android:layout_height="300dp"/>
       
设置各种属性即可
